## 🚀 Setup Locally

1. Ensure you have Git installed.
2. Clone the repo:
   ```bash
   git clone https://github.com/naveenkumarj25/Portfolio.git
   cd Portfolio
3. Open index.html in a browser or use a local server (e.g., VS Code’s Live Server extension) for the best experience.
